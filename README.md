# 📅 Calendly Clone – Full Stack Scheduling Application

A full-stack scheduling web application inspired by Calendly that enables users to create event types, define availability, and allow others to seamlessly book meetings.

---

## 🚀 Features

- Create and manage event types with customizable durations  
- Set weekly availability and time slots  
- Public booking page for scheduling meetings  
- Meeting confirmation with detailed information  
- View and manage scheduled meetings  
- Cancel meetings functionality  

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js  
- TypeScript  

**Backend:**
- FastAPI  
- SQLAlchemy  

**Database:**
- MySQL
